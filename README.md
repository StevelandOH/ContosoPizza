### .NET CLI terminal commands

- Start REPL and connect to API: \
`httprepl https://localhost:{PORT}`
- Connect to API (if REPL is already running): \
`connect https://localhost:{PORT}`
- Explore available endpoints: \
`ls`
- Select an endpoint: \
`cd {EndpointName}`
- Make a GET request: \
`get`
- Exit the REPL/CLI: \
`exit`
